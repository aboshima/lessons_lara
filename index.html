### 1. التحقق من صحة البيانات باستخدام `validate` في المتحكمات

public function store(Request $request) { $validatedData = $request->validate([ 'name' => 'required|max:255', 'email' => 'required|email|unique:users', 'password' => 'required|min:8|confirmed', ]); // إذا كانت البيانات صحيحة، يمكن تخزينها User::create($validatedData); return redirect()->back()->with('success', 'User created successfully'); }

### قواعد التحقق الشائعة في Laravel

- `max:value`: يجب ألا يزيد طول الحقل عن القيمة المحددة.
