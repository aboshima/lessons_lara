تجهيز وتثبيت البيئة:

لتثبيت **XAMPP**، وهو حزمة برمجية تتضمن Apache (خادم ويب)، MySQL (قاعدة بيانات)، وPHP (لغة برمجة)، يمكنك اتباع الخطوات التالية حسب نظام التشغيل الذي تستخدمه: ### خطوات تثبيت XAMPP: #### 1. **تنزيل XAMPP**: - قم بزيارة الموقع الرسمي لـ [XAMPP](https://www.apachefriends.org/index.html). - اختر النسخة المناسبة لنظام التشغيل الخاص بك (Windows, Linux, macOS). #### 2. **تثبيت XAMPP على نظام التشغيل الخاص بك**: ##### **على Windows:** 1. بعد تنزيل الملف التنفيذي (EXE)، افتح الملف لبدء التثبيت. 2. ستظهر نافذة التحكم في حساب المستخدم (UAC) لتحذيرك. اضغط على **Yes** للمتابعة. 3. سيظهر معالج التثبيت: - اضغط على **Next**. - اختر المكونات التي تريد تثبيتها (بشكل افتراضي، يتم تثبيت Apache وMySQL وPHP). إذا كنت لا تحتاج لبعض المكونات، يمكنك إلغاء تحديدها. - اضغط على **Next**. 4. اختر المجلد الذي تريد تثبيت XAMPP فيه (عادةً ما يكون في `C:\xampp`). 5. اضغط على **Next** ثم **Install** لبدء التثبيت. 6. بعد اكتمال التثبيت، يمكنك فتح لوحة التحكم الخاصة بـ XAMPP لبدء تشغيل خدمات Apache وMySQL. ##### **على macOS:** 1. بعد تنزيل الملف (DMG)، افتحه لبدء التثبيت. 2. اسحب أيقونة XAMPP إلى مجلد التطبيقات. 3. افتح XAMPP من مجلد التطبيقات. 4. استخدم لوحة التحكم الخاصة بـ XAMPP لتشغيل Apache وMySQL. ##### **على Linux:** 1. بعد تنزيل الملف (عادةً يكون امتداده `.run`)، افتح Terminal وانتقل إلى المجلد الذي تم فيه تنزيل الملف. 2. امنح إذنًا لتشغيل الملف: ```bash sudo chmod +x xampp-linux-x64-.run ``` 3. قم بتشغيل ملف التثبيت باستخدام الأمر التالي: ```bash sudo ./xampp-linux-x64-.run ``` 4. اتبع التعليمات الظاهرة على الشاشة لإكمال التثبيت. 5. بعد التثبيت، يمكنك بدء XAMPP باستخدام الأمر: ```bash sudo /opt/lampp/lampp start ``` #### 3. **بدء استخدام XAMPP**: - بعد تثبيت XAMPP، يمكنك فتح لوحة التحكم الخاصة به من أجل بدء خدمات مثل **Apache** و**MySQL**. - يمكنك الوصول إلى لوحة التحكم عبر: - **Windows:** افتح لوحة التحكم الخاصة بـ XAMPP من أيقونة البرنامج بعد التثبيت. - **macOS/Linux:** استخدام الأوامر أو الواجهة الرسومية حسب التثبيت. #### 4. **الوصول إلى لوحة التحكم (phpMyAdmin)**: - بمجرد تشغيل Apache وMySQL، افتح متصفح الويب واكتب العنوان التالي للوصول إلى **phpMyAdmin** وإدارة قواعد البيانات: ``` http://localhost/phpmyadmin/ ``` بذلك تكون قد أكملت تثبيت XAMPP وأصبحت جاهزًا للبدء بتطوير تطبيقاتك باستخدام PHP وMySQL.

public function create() {
return view('admin.auth.register’); }
public function store(Request $request) {
$insert = User::create([
'name' => $request->name,
'email' => $request->email,
'password' => Hash::make($request->password), ]);
if ($insert) { return redirect('login’); } }

الدخول :

public function showlogin() { return view('admin.auth.login’); }
public function login(Request $request) {
$validate = request()->validate([
'email' => 'required',
'password' => 'required’ ]);
if ($validate) {
$info = $request->only('email', 'password');
if (Auth::attempt($info))
{ return redirect()->to('dashboard’); }
return redirect()->to('login’); } }

  Laravel-mix

ميكس:

لارافيل ميكس هو حزمة تم تطويرها بواسطة جيفري واي من لاراكاست، وهي توفر واجهة برمجة تطبيقات سلسة لتعريف خطوات بناء Webpack لتطبيقات لارافيل الخاصة بك. تدعم ميكس عددًا من أنظمة CSS وJavaScript الشائعة، مما يجعل عملية إعداد وتشغيل Webpack أبسط وأكثر كفاءة.
npm install laravel-mix-tailwind --save-dev

إنشاء هذا الملف في روت المشروع
webpack.mix.js
إدراج الأكواد التالية:-
let mix = require("laravel-mix");
require("laravel-mix-tailwind");
mix.js("resources/js/app.js", "public/js")
.less("resources/less/app.less", "public/css")
.css("resources/css/app.less", "public/css")
.tailwind();
In view:-
link rel="stylesheet" href="{{ mix('resources/css/app.css') }}">
script src="{{ mix('resources/js/app.js') }}"> /script>
